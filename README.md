# ImageUploadP3

**ImageUploadP3** is a multi-screen iOS application that demonstrates a smooth onboarding flow, user input handling, image uploading, and push notification prompts. The app is built using **Swift** and **UIKit**, focusing on clean UI and simple navigation.



## About the Project

This project showcases a real-world iOS onboarding experience, starting from a welcome screen and guiding users through profile setup, image upload, and push notification enablement. It is ideal for learning UIKit navigation, user interaction, and iOS app flow design.



## Tech Stack

* **Language:** Swift
* **UI Framework:** UIKit
* **IDE:** Xcode 15+
* **Target Platform:** iOS
* **Architecture:** MVC



## App Flow & Features

### Welcome / Launcher Screen

* Initial screen displayed on app launch
* Introduces the app to the user

### User Details Screen

* Input fields for **Name** and **Username**
* Validates user input
* **Next** button navigates to the image upload screen

### Image Upload Screen

* Allows users to select an image from the device
* **Upload** button to proceed after selecting an image
* **Skip** button to continue without uploading
* Image preview support

### Push Notifications Screen

* Informs users about push notifications
* **Turn On Notifications** button
* Ready for Apple Push Notification integration



## Getting Started

### Prerequisites
